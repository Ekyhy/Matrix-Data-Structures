# Struktur Data Matrix

Struktur data matrix merupakan struktur data 2D yang menyimpan data dalam bentuk format baris dan kolom. Pada Struktur data sebuah matrix 2D terdiri dari elemen array yang memiliki tipe data yang sama. Serta memiliki jumlah ukuran baris dan kolom yang fix dan telah ditentukan sebelumnya atau disebut juga Ordo Matrix.

Program ini di buat untuk  mengimpelemtasikan operasi dasar yang dapat dilakukan pasa struktur data matrix

## Fitur Program
===================== <br>
 MENU OPERASI MATRIKS<br>
===================== <br>
1.  Sort Row-wise
2.  Sort Column-wise
3.  Rotate Clockwise (Layer by Layer)
4.  Rotate Counter-Clockwise (Layer by Layer)
5.  Rotate 90 Degrees
6.  Rotate 180 Degrees
7.  Row-wise Traversal
8.  Column-wise Traversal
9.  Spiral Traversal
10. Transpose
11. Reset ke Matriks Awal
12. Lihat Riwayat Operasi
13. Exit

## Konsep yang Dipelajari
- Struktur data array dua dimensi (2D Array)
- Representasi matrix dalam pemrograman
- Menyimpan nilai matriks sebagai bahan untuk melakukan program matriks yang lain
- Rotasi matrix
- Sorting matrix
- Traversal baris dan kolom
- Validasi ukuran (ordo) matrix
- Mencetak matrix sesuai urutan baris dan kolom
